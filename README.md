## Requirements

- [Node](https://nodejs.org/en/)
- [Gulp](https://gulpjs.com)

## Installation

1. Open terminal
2. Navigate to folder where your new project will be
3. Type:

```
git clone https://github.com/lazy-8/base.git ProjectName
cd ProjectName
npm install
bower install
gulp
```
